Q7:Write a function that accepts parameter n and returns factorial of n

Answer:const factorial=(n)=>{
    var fact = n;
    if (n === 0 || n === 1) 
      return 1; 
    while (n > 1) { 
      n--;
      fact *= n;
    }
    console.log(fact);;
}
    factorial(5)
